Release Notes for AcousticDB

dB SPL estimation software library for STM32 and ST digital MEMS microphones

Copyright 2015 STMicroelectronics

 

AcousticDB library implements a dB SPL estimation of the ambiental noise, basing on the audio signal captured by a digital MEMS microphone.


Update History

V1.0.0 / 01-Sep-2016

Main Changes

  • First official version of the library.
